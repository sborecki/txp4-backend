# txp4
Trackmania Experience v4

## Preparation

* Install [Node.js](https://nodejs.org).
* Install [yarn](https://yarnpkg.com).

## Development

Run in separate terminals:

* rebuild application on change:
```
yarn build:watch
```

* restart application on change:
```
yarn start:watch
```

* lint your code:
```
yarn lint:watch
```
